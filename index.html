What is DNS?

The Domain Name System (DNS) is the phonebook of the Internet.

What is TCP?

TCP stands for Transmission Control Protocol, a communications standard that enables application programs and computing devices to exchange messages over a network. It is designed to send packets across the internet and ensure the successful delivery of data and messages over networks.

What is an IP?

An IP address is a unique address that identifies a device on the internet or a local network. IP stands for "Internet Protocol," which is the set of rules governing the format of data sent via the internet or local network. An IP address is a string of numbers separated by periods. IP addresses are expressed as a set of four numbers — an example address might be 192.158.1.38. Each number in the set can range from 0 to 255. So, the full IP addressing range goes from 0.0.0.0 to 255.255.255.255.

What is Domain?

Think of a domain as your home address. If a friend of yours wants to visit you, how would you help him? By providing your address right? The same goes for your website. The domain allows users to identify your site. Without a domain, the user would have to enter the IP address to visit your site. So the domain name is actually a shortcut to your site. It is easy for users to remember the name rather than remembering some random numbers. Top-level domains are- .com and .org

What is HTML and Why HTML?

HTML is a markup language for hypertext (hence the name). In simple terms, HTML is a set of commands that tell web browsers how to display specific parts of websites.

HTML is the fundamental language in which most web pages are written.

Why HTML5?

HTML5 supports both video and audio. In HTML5, vector graphics are supported by default. We can draw shapes like circles, rectangles, triangles.

Async Vs Sync Code

Synchronous means to be in a sequence, i.e. every statement of the code gets executed one by one. So, basically a statement has to wait for the earlier statement to get executed.

Asynchronous code allows the program to be executed immediately where the synchronous code will block further execution of the remaining code until it finishes the current one.
